# Chassis2D
![Official logo](icon1.png "Official Logo")

A simple Game Framework based around SDL2 and Box2D to make an easy to use framework.

# Steps

To build the source code

    $mkdir build
    $cd build
    $cmake ..

Compiler will vary between operating systems
